# Ezmation

Advanced macro tool for players looking for automating from easy to hard tasks without difficulties.
What can it do?
- Autoclick and/or macro.
- Parallel mode to trigger every button simultaneously.
- Sequential mode for creating ordered triggers.
- Global and individual loop/repeat mode and delay (ms).
- Control each action's interval (ms), loop status, repeat times and PAR/SEQ.
- Separated auto clicker mode if you only need mouse automation, it includes: Interval, button, start delay, loop or repeat mode + count.
- Settings: Play and stop shortcuts, default interval and start delay times in ms, themes, "save settings" button (saves as .json file)

## Usage

```bash
python Ezmation.py
or
python3 Ezmation.py
or
Whatever your command for running Python is :)
```

Compatible with anything that can run Python.
